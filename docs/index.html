はじめに

宇宙を感じよう！

人は誰しも、一度は夜空を見上げたことがあると思います。

夜空には、月や星、惑星など、広大な宇宙が広がっています。

宇宙の広さを感じると、人間という存在は小さなものだと思い知らされます。

あなたは、宇宙を遠い存在に感じますか？

現在では、スマートフォンの地図アプリを使えば、自分の現在地を簡単に知ることができます。

地球の周りには、人類が打ち上げた人工衛星が数多く存在しています。

GPS（全地球測位システム）は、それらの人工衛星から送られる電波を利用して、現在地を割り出しています。

また、世界中の宇宙飛行士たちは、国際宇宙ステーションで宇宙に関する様々な研究を日々行っています。

近年では、宇宙旅行を体験する人々も現れ始めています。

私は、かつて遠い存在だった宇宙と、人類との距離は少しずつ近づいていると感じています。

このアプリは、宇宙空間を「感じる」ことをテーマに開発しました。

太陽系の3D表示、惑星モデル、月の満ち欠けなどを通して、宇宙をより身近に体験できます。

宇宙や天体に関心を持つ “きっかけ” になれば幸いです。
